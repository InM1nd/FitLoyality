### Ключевые боли владельцев (2025)

1. **Нехватка персонала — боль №1** независимых студий (BODYMEDIA). Наш
   sales-angle: «сотрудник, которого нельзя нанять» — автоматическая забота
   о членах без роста штата.
2. **Silent churn** — члены перестают ходить, но ещё не расторгли. Большинство
   не жалуются — просто исчезают.
3. **Первые 90 дней** — зона максимального оттока новых членов (см. §2).
4. CAC нового члена: **€50–120** маркетинговых затрат. Удержание — почти ноль.
5. Юридический фон: с 2022 (Gesetz für faire Verbraucherverträge) после
   первого срока договор расторгается помесячно → отток стал острее,
   retention-инструменты актуальнее.

## 2. Отраслевые бенчмарки удержания (ядро нашего value case)

| Показатель | Значение | Источник |
|---|---|---|
| Средний годовой retention индустрии | **66,4%** (HFA 2025) — внимание: цитируемые везде 71,4% — данные 2015 года | HFA Benchmarking Report 2025 |
| Churn бутик-студий | **35–45% в год** (= retention 55–65%) | ClubIntel Boutique Benchmark 2025 |
| Сильные бутик-операторы | retention **75–80%** | StudioPulse |
| Месячный эквивалент | 25% годовой потери ≈ **2–3% churn/мес** | Nutripy |
| **Первые 90 дней** | **50% ушедших новичков уходят в первые 90 дней** | FIA Retention Report 2025 |
| Сила онбординга | 87% 6-месячного retention у полностью онбордженных vs ~60% в контроле | Bedford research |
| **Частота → удержание** | 3+ визита/нед = **85–90%** годовой retention; 1 визит/нед = **40–50%** | отраслевые данные |

**Почему это золото для FitLoyalty:**
- Связка «частота визитов → удержание» — **прямое научное обоснование стриков**:
  каждый член, поднятый с 1 до 3 визитов/нед, почти удваивает вероятность остаться.
- «50% оттока в первые 90 дней» — обоснование Onboarding Autopilot.
- Разрыв «бутик 55–65% vs сильные 75–80%» — наше обещание: довести студию
  из среднего в сильные = 15–20 п.п. retention = десятки тысяч евро.

## 3. Тренды 2026 (куда движется наш сегмент)

- **Бутик-формат растёт:** Reformer Pilates, Lagree, Barre — «эффективно +
  эстетично»; гибридные форматы (пилатес × сила × HIIT) размывают границы.
- **EMS-сегмент: 2 443 объекта в DE (+3,6% в 2025)** — высокий чек, малые
  студии, идеальный профиль нашего клиента.
- **Боулдеринг и нишевые форматы** (outdoor bootcamps, parkour) растут;
  агрегаторные члены концентрируются на бассейнах, йоге/пилатесе и скалодромах.
- **Longevity / low-impact** — спрос на «здоровое долголетие» поднимает
  йогу/пилатес/функциональные тренировки — наш кор-сегмент.
- **Community as product:** Les Mills и Mariana Tek называют комьюнити
  главным удержателем — геймификация и лидерборды попадают в тренд.
- Corporate wellness продолжает расти (см. §6).

**Вывод:** все растущие форматы — классовые бутик-студии = наша ЦА расширяется
сама по себе.

Источники: [fitnessmanagement.de — 7 Trends 2026](https://www.fitnessmanagement.de/fitnesstrends-2026-les-mills-fitnessstudios/), [Mariana Tek — Boutique Trends 2026](https://www.marianatek.com/blog/6-trendsetting-boutique-fitness-studios-to-watch-in-2026/), [bsport — Trends 2026](https://pro.bsport.io/en/blog/fitness-trends-shaping-boutique-studios-in-2026), [Deloitte — Fitnessmarkt](https://www.deloitte.com/de/de/Industries/consumer/research/deutscher-fitnessmarkt-studie.html)

## 4. Софт-ландшафт: кто чем пользуется

### Magicline / Sport Alliance — лидер, но НЕ наш сегмент

- **~10 000 клиентов**, ядро — сети: McFIT, clever fit, EasyFitness, bodystreet.
- **Цены (важно для нашего позиционирования):** 5 пакетов от Free до €699/мес
  за локацию. Free — до 25 членов; **Basic €189/мес** (до 500 членов);
  **Advanced €349/мес** (до 1 500 членов, первая партнёрская интеграция).
- **Уже строит retention сам:**
  - Программа лояльности (награды за чек-ины, уровни) — в тарифе Advanced (€349!).
  - **MagicAI Engage** (в пилоте, «bald verfügbar»): ML-предсказание оттока,
    7 lifecycle-моментов, WhatsApp/SMS/email/push, двусторонние AI-диалоги.
    Заточен под сети/мультилокации.
  - Чего у них НЕТ: member-facing геймификации, стриков, white-label
    приложения для члена клуба.
- **Стратегический вывод: клиентов Magicline НЕ таргетируем** — они получат
  retention нативно. Но заметь: их retention живёт в пакете за €349/мес —
  наш Growth €149 дешевле вдвое и с member-app.

Источники: [Magicline Pricing](https://www.magicline.com/en/pricing/), [OneClickGym — Alternativen](https://oneclickgym.de/magicline-alterantiven/), [MagicAI Engage](https://www.magicline.com/de/magic-ai/engage/), [BODYMEDIA — Kundenbindungsprogramm](https://www.bodymedia.de/news/magicline-integriert-kundenbindungsprogramm-in-die-software/)

### Eversports Manager — софт нашего кор-клиента

- **~4 000 клиентов**: йога, пилатес, танцы, бутик-студии. Домашний рынок — Австрия.
- **Цены: от €49/мес** (Light €59/мес до 49 бронирований; скейлится по классам
  и членам) — то есть наш Starter €49 = цена их booking-софта, психологически
  «вторая такая же подписка».
- Retention-функции: только email-автоматизация, сегменты, промокоды.
  **Нет churn-prediction, нет геймификации, нет branded member app — наша дыра.**
- **Provider API (критично!):** GraphQL, **отдаёт ТОЛЬКО расписания**.
  Customer/Booking/Financial Data — явно исключены. €50/мес за локацию,
  read-only. → Автосинхронизация посещаемости через официальный API
  **невозможна** на сегодня.
- Что есть вместо API:
  - **CSV-экспорт списков участников** (Teilnehmerliste) — наш путь в MVP.
    Недельной гранулярности достаточно для churn-детекции.
  - Отметка присутствия — **опциональна** → шум no-show. Fallback-сигнал:
    «записался + не отменил ≈ пришёл».
  - **Market extensions** — партнёрская экосистема (USC, Wellhub, ClassPass,
    MyClubs, BODDY, Workit, EGYM Wellpass, Zenspotting). Глубокие интеграции
    существуют → BD-путь, когда будут платящие студии.
- Риск: Eversports может построить лояльность сам. Митигация: собственный
  QR-слой + отношения с членом через app — это у нас не отнять.

### Как реально происходит чек-ин в классовых студиях

- **Карточек и турникетов НЕТ.** Член бронирует класс в Eversports → тренер

- Sales-line: «Твои тренеры уже тикают присутствие — мы превращаем эти галочки
  в удержание и выручку. Ноль изменений в процессах».

## 6. Агрегаторы и рынок Firmenfitness — «большая четвёрка»

Ключевой структурный фактор рынка DACH. Типичная студия подключена к 2–3 сразу.

### Рынок корпоративного фитнеса в целом

- Corporate-wellbeing рынок: прогноз роста до **$106 млрд к 2029**, CAGR ~4,1%.
- Wellpass, Hansefit, USC растут **двузначными темпами**.
- Roland Berger: компании дают в среднем **6,6 бенефитов** и планируют +1,4 в 2025 —
  фитнес-бенефит стал гигиеническим фактором найма.
- **Wellhub-студия рынка (валидация нашего share-of-wallet контура):**
  97% студий-партнёров видят более высокое удержание у членов, пришедших
  через Firmenfitness; 98% — рост рентабельности; 93% считают канал
  эффективным для привлечения.

### EGYM Wellpass — лидер корпоративного фитнеса DE

- **13 950 партнёрских студий** на конец 2025 (**+3 050 за год**)
- **690 000+ сотрудников** из **21 000+ компаний**
- **Corporate-only** → конвертировать Wellpass-визитёра в прямое членство
  **невозможно в принципе**
- От ~€25/сотрудник/мес; премиум-сеть: Fitness First, INJOY, EVO, Just Fit,
  Therme Erding, BEAT81. Владелец — EGYM (Мюнхен, тренажёры + софт).

### Urban Sports Club / Wellhub

- **~10 000+ партнёров в Германии** (в 2023 — ~6 000), 7 стран Европы
- **Март 2025: Wellhub (ex-Gympass) купил USC.** Объединённо: ~30 000 залов
  в Европе, 83 000 wellness-партнёров глобально, ~50 000 корп. клиентов,
  **5+ млн подписчиков-сотрудников**
- **Единственный агрегатор с self-pay членами** — но корпоративные = ~45%
  новых регистраций и ~45% выручки 2025, доля растёт
- Частные тарифы ~€39–129/мес; корп.: €28 сервис-сбор + субсидия €30/40/49,90;
  работодатель — до **€50/мес tax-free**
- **Выплата студии: ~€8 за визит** против €80–150 прямого членства
- **Зависимость студий — системная боль:** до 99% бронирований через USC;
  «расторжение = конец бизнеса» (Tagesspiegel, WiWo, taz)

### Hansefit

- **8 500+ партнёров**, corporate-only, бюджетный (~€30/сотрудник), фокус
  на классические студии

### Что это значит для FitLoyalty (Aggregator Hub)

1. **Share-of-wallet (массовый контур):** за каждый визит агрегаторного гостя
   студия получает ~€4–8. Стрики заставляют выбирать именно эту студию →
   больше выплат. Работает на 100% агрегаторного трафика, включая
   corporate-only Wellpass и Hansefit. Подтверждено Wellhub-студией: канал
   уже воспринимается студиями как прибыльный — мы делаем его измеримым и растущим.
2. **Revenue-mix visibility:** владелец жонглирует 2–3 порталами и не видит
   общей картины. Одна панель: direct vs USC vs Wellpass vs Hansefit. Нет ни у кого.
3. **Конверсия — только USC self-payers.** Честная математика: из 100
   USC-визитёров ~50–60 корпоративных (неконвертируемы), из остальных лояльны
   одной студии 20–30% → пул 10–15 → **1–3 конверсии за кампанию**,
   каждая +€80–150 MRR.

## 7. Юридика и комплаенс (DACH)

### WhatsApp / мессенджер-маркетинг

- **§7 Abs. 2 UWG:** реклама в мессенджерах — только с явным предварительным
  согласием. Исключение для Bestandskunden покрывает **email, не мессенджеры**.
- Требования: **double opt-in** с таймстемпом, документированное;
  сертифицированный WhatsApp BSP; **AVV**.
- → Продуктовое решение: согласия собираются при онбординге в member app.
  «Consent Manager: DSGVO-konform из коробки» — сама по себе фича
  (владельцы боятся Abmahnungen).

### DSGVO-роли

- Студия = **controller**, FitLoyalty = **processor** → шаблон AVV с каждой
  студией, EU-хостинг, чистый список субпроцессоров.
- Посещаемость — НЕ Art. 9, но держим zero-knowledge позицию: никакой
  биометрии/пульса, только визиты.

### USC партнёрский договор

- **Abwerbeverbot вероятен, публично не найден — ПРОЧИТАТЬ ДОГОВОР до
  постройки конверсионной части.** Безопасный дизайн: член добровольно ставит
  приложение студии → согласие напрямую студии → офферы in-app.

### Kündigungsfrist — наше DACH-преимущество

- Реальное окно спасения — **дедлайн уведомления о расторжении**, не дата
  окончания договора. Алерты привязываем к нему. US-конкуренты этого не понимают.

Источники: [IT-Recht-Kanzlei — WhatsApp](https://www.it-recht-kanzlei.de/whatsapp-newsletter-versand-rechtliche-umsetzung-dsgvo.html), [IHK Stuttgart — §7 UWG](https://www.ihk.de/stuttgart/fuer-unternehmen/recht-und-steuern/wettbewerbsrecht/richtig-werben/was-ist-erlaubt-684868), [werkvonmorgen — WhatsApp DSGVO](https://www.werkvonmorgen.de/blog/whatsapp-marketing-dsgvo-so-vermeidest-du-rechtliche-risiken)

## 11. Юнит-экономика

### Для студии (value case — используем в питче и Churn-Check)

Типовая студия: 220 членов × €79 = **€17 380 MRR**.
- Бутик-churn 35%/год → ~77 уходов/год → **~€73 000** потерянной годовой выручки.
- FitLoyalty Growth: €1 788/год.
- Спасение хотя бы 10% уходящих (8 чел. × €79 × 12) = **€7 584/год** → ROI ≈ 4x.
- Плюс агрегаторный контур: 300 агрегаторных визитов/мес × ~€7 = €2 100/мес;
  +10% частоты от стриков = **+€210/мес** — этот контур сам по себе окупает Growth.
- Break-even: **2 спасённых членства в месяц.**

### Для нас (SaaS)

- Цель 3 года: 300 студий × blended €120 = **€36k MRR / ~€430k ARR**.
- Target CAC ≤ €400–500 (self-serve воронка Churn-Check + контент) →
  payback 3–4 месяца.
- Главный риск экономики — **churn самих студий**: при чеке €49–149
  white-glove onboarding невозможен → self-serve обязателен, Saved Revenue
  должен быть честным (иначе отток на первом продлении).
- Расширение: тиры (Starter→Growth при подключении агрегаторов), мультилокация
  (Pro), в перспективе — сетевые бенчмарки как add-on.

**Threats**

- Eversports построит loyalty сам (у него и данные, и биллинг)
- Агрегаторы запустят собственную retention-программу для партнёров
- Рост цен WhatsApp Business API; ужесточение consent-практики

## 14. Стратегические выводы (зафиксированы в .claude/CLAUDE.md)

### Кор-клиент

Owner-operated студия **100–500 членов** (йога/кроссфит/бокс/EMS/боулдеринг),
сидит на **Eversports/bsport/fitogram или Excel+WhatsApp**, часто зависит от
агрегаторов. **НЕ клиенты Magicline.** Beachhead: **Австрия** (1 349 студий,
домашний рынок Eversports, основатель в Вене).

### Позиционирование

Не «AI churn prediction» (это поле заберёт MagicAI Engage), а:
**white-label loyalty app для членов + измеримый возврат денег (спасённые
члены + агрегаторная экономика) для студий, которым Magicline велик, а
Keepme не по карману.**

> «Eversports runs your bookings. We bring your revenue back.»

### Питч-цифры

- Один спасённый член ≈ окупает Starter; два — Growth.
- Churn-Check: «€X MRR под угрозой прямо сейчас» за 5 минут, бесплатно.
- Сейлз-якорь №2: нехватка персонала → «автопилот заботы о членах».
- Научная база: 3+ визита/нед = 85–90% retention (стрики), 50% оттока в
  первые 90 дней (Onboarding Autopilot).

### Ключевые фичи и их рыночное обоснование

| Фича | Обоснование из рынка |
|---|---|
| Weekly streaks (не daily!) + freeze | люди ходят 2–3×/нед; частота → retention (85–90% vs 40–50%) |
| Churn-window алерты по Kündigungsfrist | немецкая договорная механика, US-конкуренты не понимают |
| Saved Revenue counter | владелец продлевает, когда видит €€, а не heatmap |
| Aggregator Hub | 3 контура; Wellhub-студия подтверждает прибыльность канала |
| Live occupancy | нет ни у одного бутик-софта; драйвер ежедневного открытия app |
| Review & Referral Engine | 41% конверсия рефералов; ~1 отзыв/день |
| Onboarding Autopilot (90 дней) | 50% оттока в первые 90 дней; 87% vs 60% retention |
| Win-back | бывшие члены — тёплая DSGVO-легальная база |
| Consent Manager | §7 UWG + страх Abmahnung = фича |
| CSV-импорт + QR (не API!) | Eversports API не отдаёт посещаемость |

### Главные риски

1. **Платформенные:** MagicAI Engage; Eversports может сам. Митигация:
   QR-слой + app-отношения с членом + мультиисточниковость.
2. **Данные:** нет API посещаемости; опциональные галочки → no-show шум.
3. **Юридический:** USC Abwerbeverbot (проверить договор), WhatsApp opt-in.
4. **GTM:** SMB-SaaS churn — честная атрибуция + self-serve обязательны.
5. **Сезонность:** январь раздут, лето проседает → сравнения год-к-году.
